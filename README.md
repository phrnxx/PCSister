# 💜 PCSister - Твій автономний десктопний компаньйон

PCSister - це легковажний десктопний додаток на базі **C# (WPF)**, який розміщує на вашому робочому столі чарівну чибі-дівчинку. Вона живе поверх вікон, реагує на дії користувача, читає буфер обміну, відчуває навантаження ПК і дізнається запущені ігри.

---

## ✨ Основні особливості

### 🧠 Жива поведінка та Автономність
- **Своє життя:** Персонаж не стоїть на місці - воно дихає, гуляє по панелі завдань, стрибає і іноді "замислюється".
- **Сплячий режим:** Якщо курсор миші не рухається більше 5 хвилин, вона лягає спати, щоб не відволікати вас.

### 🏃 Механіка «Спринт і Невдача»
- Кожні 1-3 хвилини персонаж може раптово почати бігати екраном.
- Наприкінці пробіжки є 50% шанс, що вона спіткнеться та впаде (анімація `Dead.png`).
- Після падіння вона обов'язково висловлює невдоволення у чаті (наприклад, скаржиться на гравітацію).

### 🔥 Розумні функції (Hardware & Context)
- **Залізна емпатія (CPU Monitor):** Якщо навантаження на ваш процесор перевищує 85%, воно починає панікувати, бігати і кричати про "Викид" або перегрівання.
- **Цифровий Барабашка (Clipboard):** Реагує на те, що ви копіюєте в буфер обміну (коментує посилання на YouTube, StackOverflow або скопійовані помилки коду).
- **Режим Фокусу:** Вмикається через контекстне меню. Якщо ви відкриєте відволікаючу програму (Discord, TikTok, Steam і т.д.) під час роботи, вона почне на вас лаятися.

### ⚙️ Системна інтеграція (WPF)
- **Ідеальний Click-Through:** Вікно повністю прозоре для миші. Ви можете вільно клацати за своїм кодом або браузером крізь порожні області навколо персонажа.
- **Topmost:** Нативний WinAPI-рендер завжди тримає її поверх інших вікон (VS Code, Chrome і т.д.), не конфліктуючи із системою.

## 🛠 Управління

Взаємодія з PCSister відбувається безпосередньо через саму фігурку:

| Дія | функція |
|--------------------|--------|
| **ПКМ (Правий клік)** за персонажем | Відкриває контекстне меню
| У меню: Режим фокусу | Включає/вимикає контроль відволікаючих програм
| У меню: `Віхід` | Закриває програму |
| Різке рух мишею | Дівчинка лякається (`Hurt` анімація)

## 🚀 Встановлення та запуск

### 📦 Для користувачів
1. Завантажте архів з останньою версією розділу **Releases**.
2. Розпакуйте у зручне місце та запустіть `PCSister.exe`.

---

### 🧑‍💻 Для розробників (C# / WPF)

Проект був переписаний з Python (Pygame) на C# (.NET WPF) для ідеальної інтеграції з Windows DWM і забезпечення прозорості кліків.

**Вимоги:**
* Visual Studio 2022 (або новіше)
* .NET 6.0 Desktop Runtime (або новіше)

**Складання:**
1. Відкрийте `PCSister.sln` у Visual Studio.
2. Переконайтеся, що всі файли в папці `Assets/` мають *Build Action (Дія при складанні)* встановлена ​​як **Resource**.
3. Зберіть проект (`Ctrl+Shift+B`) або запустіть (`F5`).
